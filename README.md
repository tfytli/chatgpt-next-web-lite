# NeatChat
NeatChat是NextChat的去广告版本，使用NextChat的UI，并移除了广告。
更改了配色，去除了多语言支持，只保留中文和英文，只为精简。

详细使用请参考[NextChat](https://github.com/ChatGPTNextWeb/ChatGPT-Next-Web)
